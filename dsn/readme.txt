Archivos de diseño
==================

Encuentra en esta carpeta los archivos de diagramas eléctricos para el
armado del módulo de tranmisión de datos.

Archivos de diseño principal:

* main\Enlaza.sch : Archivo fuente del diagráma eléctrico para Eagle.
* main\Enlaza.brd : Archivo fuente de la tableta impresa para Eagle.

Archivos de diseño auxiliar:
* auxiliar\Enlaza.pdf : PDF para impresión del diagrama eléctrico.
